# /security-dependency-audit

Audits dependencies for security vulnerabilities.

## Usage

```bash
/security-dependency-audit [--fix]
```

## Checks

- Known CVEs in packages
- Outdated dependencies
- Malicious packages
- License compliance
- Supply chain risks
- Transitive vulnerabilities

## Example

```bash
/security-dependency-audit --fix
```
