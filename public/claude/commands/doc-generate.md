# /doc-generate

Generates comprehensive documentation.

## Usage

```bash
/doc-generate [--type=api|component|contract]
```

## Generates

- API reference docs
- Component documentation
- Architecture diagrams
- Usage examples
- Smart contract docs (Web3)
- Deployment guides

## Example

```bash
/doc-generate --type=api
```
