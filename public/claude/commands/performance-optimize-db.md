# /performance-optimize-db

Optimizes database queries and schema.

## Usage

```bash
/performance-optimize-db [--analyze-slow-queries]
```

## Optimizations

- Index recommendations
- Query plan analysis
- N+1 query detection
- Connection pool tuning
- Schema normalization
- Blockchain data indexing (Web3)

## Example

```bash
/performance-optimize-db --analyze-slow-queries
```
