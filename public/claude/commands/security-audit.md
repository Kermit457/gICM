# /security-audit

Performs comprehensive security analysis.

## Usage

```bash
/security-audit [--deep]
```

## Scans For

- SQL injection vulnerabilities
- XSS attack vectors
- CSRF protection gaps
- Authentication weaknesses
- Authorization flaws
- Reentrancy attacks (Web3)
- Integer overflow (Solidity)

## Example

```bash
/security-audit --deep
```
