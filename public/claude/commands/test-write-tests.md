# /test-write-tests

Develops unit and integration tests for existing code.

## Usage

```bash
/test-write-tests <file_path> [--framework=jest|mocha|hardhat]
```

## Features

- Automatic test scaffold
- Mocking setup
- Assertions generation
- Coverage targets

## Example

```bash
/test-write-tests src/utils/bonding-curve.ts --framework=jest
```
