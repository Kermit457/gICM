# /refactor-clean

Removes unused code and dependencies.

## Usage

```bash
/refactor-clean [--dry-run]
```

## Removes

- Unused imports
- Dead code paths
- Deprecated functions
- Redundant dependencies
- Commented-out code
- Orphaned files

## Example

```bash
/refactor-clean --dry-run
```
