# Brave Search MCP Server

Web and local search capabilities with privacy focus.

## Installation

```bash
npx gicm-stack add mcp/brave-search
```

## Environment Variables

```bash
BRAVE_API_KEY=your_api_key
```

## Features

- Web search
- Local search
- Privacy-focused results
- No tracking

## Tools

- `brave_search` - Search the web
- `brave_local_search` - Search locally
