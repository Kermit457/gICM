# CircleCI MCP Server

Enable AI agents to fix CircleCI build failures automatically.

## Installation

```bash
npx gicm-stack add mcp/circleci
```

## Environment Variables

```bash
CIRCLECI_API_TOKEN=your_api_token
CIRCLECI_ORG_ID=your_org_id
```

## Features

- Build failure analysis
- Pipeline management
- Automated debugging
- Workflow orchestration

## Tools

- `circleci_get_builds` - List builds
- `circleci_analyze_failure` - Debug failures
- `circleci_trigger_pipeline` - Start builds
