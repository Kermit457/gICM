# AWS Bedrock MCP Server

Query Amazon Bedrock Knowledge Bases for AI-powered search and retrieval.

## Installation

```bash
npx gicm-stack add mcp/aws-bedrock
```

## Environment Variables

```bash
AWS_ACCESS_KEY_ID=your_access_key
AWS_SECRET_ACCESS_KEY=your_secret_key
AWS_REGION=us-east-1
BEDROCK_KNOWLEDGE_BASE_ID=your_kb_id
```

## Features

- Knowledge base queries
- Vector search
- Document retrieval
- RAG applications

## Tools

- `bedrock_query` - Query knowledge base
- `bedrock_retrieve` - Retrieve documents
