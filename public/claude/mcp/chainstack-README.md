# Chainstack MCP Server

Connect AI models to Ethereum-compatible networks with enhanced RPC access.

## Overview

Professional blockchain RPC access through Chainstack's infrastructure. Access live documentation, smart contract storage, and on-chain state.

## Installation

```bash
npx gicm-stack add mcp/chainstack
```

## Environment Variables

```bash
CHAINSTACK_API_KEY=your_api_key
CHAINSTACK_ENDPOINT=https://nd-123-456-789.p2pify.com/your_key
```

## Features

- Enterprise-grade RPC endpoints
- Smart contract state queries
- Block and transaction data
- Event log filtering
- Historical data access

## Tools

- `chainstack_get_balance` - Get wallet balance
- `chainstack_get_block` - Fetch block data
- `chainstack_call_contract` - Read contract state
- `chainstack_get_logs` - Query event logs
