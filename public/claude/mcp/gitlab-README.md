# GitLab MCP Server

Official GitLab integration with CI/CD pipeline support.

## Installation

```bash
npx gicm-stack add mcp/gitlab
```

## Environment Variables

```bash
GITLAB_TOKEN=your_personal_access_token
GITLAB_URL=https://gitlab.com
```

## Features

- Merge request management
- CI/CD pipeline triggers
- Issue tracking
- Code reviews
- Repository operations

## Tools

- `gitlab_create_mr` - Create merge requests
- `gitlab_trigger_pipeline` - Run CI/CD
- `gitlab_manage_issues` - Issue operations
