# Docker MCP Server

Manage Docker containers with natural language commands.

## Installation

```bash
npx gicm-stack add mcp/docker
```

## Features

- Container management (start, stop, remove)
- Image building
- Network configuration
- Volume management

## Tools

- `docker_run` - Run containers
- `docker_build` - Build images
- `docker_ps` - List containers
- `docker_logs` - View logs
