# Stripe MCP Server

Interact with Stripe payment API for payment processing.

## Installation

```bash
npx gicm-stack add mcp/stripe
```

## Environment Variables

```bash
STRIPE_SECRET_KEY=sk_live_...
```

## Features

- Payment operations
- Subscription management
- Customer handling
- Invoicing
- Webhook handling

## Tools

- `stripe_create_payment` - Process payments
- `stripe_manage_subscription` - Subscriptions
- `stripe_get_customer` - Customer data
