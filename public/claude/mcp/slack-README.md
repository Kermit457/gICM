# Slack MCP Server

Slack workspace interaction and messaging automation.

## Installation

```bash
npx gicm-stack add mcp/slack
```

## Environment Variables

```bash
SLACK_BOT_TOKEN=xoxb-...
SLACK_WORKSPACE_ID=T...
```

## Features

- Messaging
- Channel management
- Reactions
- User profiles
- Thread management

## Tools

- `slack_send_message` - Send messages
- `slack_create_channel` - Create channels
- `slack_get_threads` - Read threads
