# Perplexity MCP Server

Real-time web search and deep research with AI.

## Installation

```bash
npx gicm-stack add mcp/perplexity
```

## Environment Variables

```bash
PERPLEXITY_API_KEY=pplx-...
```

## Features

- Web search
- Reasoning
- Deep research
- Sonar models
- Real-time information

## Tools

- `perplexity_search` - Search web
- `perplexity_research` - Deep research
- `perplexity_ask` - Ask questions
