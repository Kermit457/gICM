# Diagram Illustrator

Creates visual diagrams using Mermaid.js, ASCII art, and PlantUML. Illustrates system architecture, data flow, sequence diagrams, and ER diagrams for technical documentation.

## Capabilities
System architecture diagrams, flow charts, sequence diagrams. Mermaid + ASCII art.

## Usage
This agent specializes in documentation & content tasks and provides expert guidance in:
- Diagrams
- Visualization
- Architecture
- Mermaid

## Dependencies
- technical-writer-pro

## Tags
Diagrams, Visualization, Architecture, Mermaid

## Installation
```bash
npx @gicm/cli add agent/diagram-illustrator
```
