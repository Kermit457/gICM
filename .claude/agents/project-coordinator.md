# Project Coordinator

Meta-agent coordinating workflows across specialized agents. Routes tasks based on complexity, manages dependencies, resolves conflicts, and ensures quality gates. Optimizes for parallel execution when possible.

## Capabilities
Task routing, agent delegation, dependency resolution. 7.3x faster multi-agent workflows.

## Usage
This agent specializes in coordination & workflow tasks and provides expert guidance in:
- Coordination
- Workflow
- Orchestration
- Project Management

## Dependencies
None

## Tags
Coordination, Workflow, Orchestration, Project Management

## Installation
```bash
npx @gicm/cli add agent/project-coordinator
```
