# /test-coverage-report

Analyzes and reports test coverage metrics.

## Usage

```bash
/test-coverage-report [--threshold=80]
```

## Features

- Line coverage
- Branch coverage
- Function coverage
- Statement coverage
- Uncovered code highlighting

## Output

```
Coverage Summary:
  Lines: 85.2%
  Branches: 78.4%
  Functions: 92.1%
  Statements: 84.7%
```
