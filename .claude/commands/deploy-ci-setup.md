# /deploy-ci-setup

Establishes CI/CD pipelines.

## Usage

```bash
/deploy-ci-setup [--platform=github|gitlab|circle]
```

## Configures

- Build stages
- Test automation
- Deploy stages
- Environment variables
- Secrets management

## Example

```bash
/deploy-ci-setup --platform=github
```
