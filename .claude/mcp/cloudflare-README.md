# Cloudflare MCP Server

Deploy and configure Cloudflare resources: Workers, R2, DNS, CDN.

## Installation

```bash
npx gicm-stack add mcp/cloudflare
```

## Environment Variables

```bash
CLOUDFLARE_API_TOKEN=your_api_token
CLOUDFLARE_ACCOUNT_ID=your_account_id
```

## Features

- Workers deployment
- R2 storage management
- DNS configuration
- CDN settings
- DDoS protection

## Tools

- `cloudflare_deploy_worker` - Deploy serverless functions
- `cloudflare_manage_dns` - Update DNS records
- `cloudflare_configure_cdn` - CDN settings
