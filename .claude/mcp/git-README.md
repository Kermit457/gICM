# Git MCP Server

Core Git operations for repository manipulation.

## Installation

```bash
npx gicm-stack add mcp/git
```

## Features

- Repository reading
- Code search
- Commit history
- Branch management
- Diff analysis

## Tools

- `git_log` - View commit history
- `git_diff` - Compare changes
- `git_search` - Search code
- `git_branch` - Branch operations
