# Firecrawl MCP Server

Extract web data with advanced crawling capabilities.

## Installation

```bash
npx gicm-stack add mcp/firecrawl
```

## Environment Variables

```bash
FIRECRAWL_API_KEY=fc-...
```

## Features

- Web scraping
- Content extraction
- Structured data parsing
- Site crawling

## Tools

- `firecrawl_scrape` - Scrape URL
- `firecrawl_crawl` - Crawl site
- `firecrawl_extract` - Extract data
