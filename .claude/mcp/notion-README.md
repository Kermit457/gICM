# Notion MCP Server

Official Notion integration for workspace management.

## Installation

```bash
npx gicm-stack add mcp/notion
```

## Environment Variables

```bash
NOTION_API_KEY=secret_...
```

## Features

- Page management
- Database queries
- Content creation
- Workspace organization

## Tools

- `notion_create_page` - Create pages
- `notion_query_database` - Query data
- `notion_update_page` - Update content
