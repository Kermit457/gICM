# Google Cloud Run MCP Server

Deploy containerized applications to Google Cloud Run.

## Installation

```bash
npx gicm-stack add mcp/google-cloud-run
```

## Environment Variables

```bash
GOOGLE_APPLICATION_CREDENTIALS=/path/to/service-account.json
GCP_PROJECT_ID=your_project_id
```

## Features

- Container deployment
- Service management
- Auto-scaling configuration
- Traffic splitting

## Tools

- `cloudrun_deploy` - Deploy containers
- `cloudrun_list_services` - List services
- `cloudrun_scale` - Configure scaling
