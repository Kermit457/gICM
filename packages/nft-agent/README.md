# @gicm/nft-agent

> NFT operations agent - Metaplex, OpenSea, Magic Eden integration

[![npm version](https://badge.fury.io/js/@gicm%2Fnft-agent.svg)](https://www.npmjs.com/package/@gicm/nft-agent)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Installation

```bash
npm install @gicm/nft-agent
# or
pnpm add @gicm/nft-agent
```

## Features

- **Metaplex**: Solana NFT minting and management
- **OpenSea**: EVM NFT marketplace operations
- **Magic Eden**: Solana marketplace integration
- **Collection Analysis**: Floor price, volume tracking
