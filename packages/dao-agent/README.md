# @gicm/dao-agent

> DAO governance agent - Snapshot, Tally, Realms integration

[![npm version](https://badge.fury.io/js/@gicm%2Fdao-agent.svg)](https://www.npmjs.com/package/@gicm/dao-agent)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Installation

```bash
npm install @gicm/dao-agent
# or
pnpm add @gicm/dao-agent
```

## Features

- **Snapshot**: Off-chain voting and proposals
- **Tally**: On-chain governance for EVM DAOs
- **Realms**: Solana DAO governance
- **Proposal Tracking**: Monitor and vote on proposals
