# @gicm/deployer-agent

> Deployment agent - Publishes to registry, npm, and GitHub

[![npm version](https://badge.fury.io/js/@gicm%2Fdeployer-agent.svg)](https://www.npmjs.com/package/@gicm/deployer-agent)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Installation

```bash
npm install @gicm/deployer-agent
# or
pnpm add @gicm/deployer-agent
```

## Features

- **npm Publishing**: Automated package publishing
- **GitHub Releases**: Create releases with changelogs
- **Registry Sync**: Sync to gICM marketplace
- **Version Bumping**: Semantic versioning
