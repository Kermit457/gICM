# @gicm/refactor-agent

> Code quality and refactoring agent

## Installation

```bash
npm install @gicm/refactor-agent
# or
pnpm add @gicm/refactor-agent
```

## Features

- **Code Analysis**: Identify refactoring opportunities
- **Pattern Matching**: Apply consistent patterns
- **Type Improvements**: Enhance TypeScript types
- **Dead Code Removal**: Clean unused code
