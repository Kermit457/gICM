# @gicm/bridge-agent

> Cross-chain bridge agent - Wormhole, LayerZero, DeBridge integration

[![npm version](https://badge.fury.io/js/@gicm%2Fbridge-agent.svg)](https://www.npmjs.com/package/@gicm/bridge-agent)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Installation

```bash
npm install @gicm/bridge-agent
# or
pnpm add @gicm/bridge-agent
```

## Features

- **Wormhole**: Multi-chain message passing
- **LayerZero**: Omnichain interoperability
- **DeBridge**: Cross-chain swaps and transfers
- **Route Optimization**: Find best bridge paths
