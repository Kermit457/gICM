# @gicm/social-agent

> Social media agent - Twitter, Telegram, Discord, Farcaster integration

[![npm version](https://badge.fury.io/js/@gicm%2Fsocial-agent.svg)](https://www.npmjs.com/package/@gicm/social-agent)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Installation

```bash
npm install @gicm/social-agent
# or
pnpm add @gicm/social-agent
```

## Features

- **Twitter/X**: Post, reply, monitor trends
- **Telegram**: Bot and channel management
- **Discord**: Server automation and moderation
- **Farcaster**: Decentralized social posting
